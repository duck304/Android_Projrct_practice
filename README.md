# Hackathon & Team Project Practice

이 저장소는 해커톤과 팀 프로젝트를 대비하기 위해 만든 연습 공간입니다.
GitHub 사용법, 협업 흐름, 브랜치 관리, 커밋 메시지 작성, Pull Request 연습 등을 자유롭게 시도하며 익히는 것을 목표로 합니다.

## 목적

* GitHub 협업 방식에 익숙해지기
* 팀 프로젝트 진행 전 기본 개발 흐름 연습하기
* 브랜치 생성, 커밋, 푸시, Pull Request 과정 익히기
* 해커톤에서 빠르게 개발하고 협업할 수 있는 환경 만들기

## 기대 효과

* 팀원들과의 코드 공유 및 협업 능력 향상
* Git과 GitHub 사용에 대한 자신감 증가
* 프로젝트 진행 중 발생할 수 있는 충돌 상황 미리 경험
* 역할 분담, 기능 구현, 코드 리뷰 흐름 이해
* 실제 해커톤 및 팀 프로젝트에서 더 효율적으로 작업 가능

## Practice List

## Tech Stack

`Git` `GitHub` `Markdown`

## Note

이 공간은 완성된 프로젝트보다 **협업 과정과 GitHub 사용법을 연습하는 것**에 초점을 둡니다.
작은 실수도 경험으로 삼으며, 실제 프로젝트에서 더 좋은 팀워크를 만들기 위한 준비 공간입니다.
